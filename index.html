Shebang Bot is an IRC bot for those who want a bot for everything. It provides easy searching via popular internet search engines (such as Google, Yahoo, Bing, and DuckDuckGo), easy communication via Twilio, channel history, food recommendations via Yelp, weather, stock quotes, definitions/synonyms, news headlines, user location information via Foursquare, user music preferences via Last.fm, and much more (the whole shebang).

Shebang bot will be open sourced upon release. It will be written in Node.js and primarily utilize the Jerk npm package. It will then utilize a number of 3rd party apis to provide the majority of the functionality. These apis include Google, Amazon, Facebook, Bing, Yahoo, Twilio, Foursquare, Last.fm, Twitter, etc. For storage, the bot will use SQLite. However, it will be able to swap in a more heavy duty database if necessary.

FAQ

Why does the world need another IRC bot?
It probably doesn't. All of this functionality exists in a bot somewhere. I'm just trying to encapsulate it into a single bot and make it perform great.
